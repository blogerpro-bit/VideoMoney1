# VidéoMoney - Plateforme de Monétisation Vidéo

Une plateforme où les créateurs gagnent 0,5 XAF par vue et retirent via Orange Money / MTN Mobile Money.

## Déploiement
1. Va sur https://vercel.com
2. Clique "Add New" → "Deploy without Git"
3. Glisse tous les fichiers
4. Modifie la clé NotchPay dans script.js

## NotchPay
- Crée un compte sur https://business.notchpay.co
- Va dans Settings → Developer → API Keys
- Copie ta clé publique (pk_test_...)
- Remplace dans script.js
